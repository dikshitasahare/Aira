# 🤖 AIRA – A Full-Stack AI Assistant Built from Scratch

AIRA is a full-stack AI assistant built with **Django** that delivers intelligent conversations, real-time streaming responses, document analysis, code assistance, and productivity tools through a modern, responsive web interface.

## 🚀 Overview

The goal of this project was to understand how conversational AI applications work beyond simply integrating AI APIs. AIRA combines real-time communication, intelligent document processing, conversation management, and developer-focused tools into a seamless user experience.

---

## ✨ Features

### 💬 Smart AI Chat

* Real-time streaming responses
* Context-aware conversations (last 20 messages)
* Automatic conversation title generation
* Pin, rename, and delete conversations
* Markdown rendering
* Syntax-highlighted code blocks
* One-click code copy

### 📄 Document & File Understanding

* PDF explanation
* Image analysis
* Word (.docx) document understanding
* Code file analysis (.py, .js, .html, etc.)
* Text file support (.txt, .csv, .md)
* Side-by-side document comparison

### 🛠️ Built-in AI Tools

* Calculator
* Weather information (OpenWeather API)
* Python code execution sandbox

### 🎨 Modern User Experience

* Token-by-token streaming responses
* Edit sent messages
* Copy AI responses
* Mobile-responsive design
* Dark mode interface
